val add : ?merge:('a -> 'a -> 'a) -> int -> 'a -> 'a t -> 'a t
add ~merge:f k d m returns a map containing the same bindings as m, plus a binding of k to d. If k was already bound to d' in m, then the value f d' d is added instead of d. If no merge function is specified, then the previous bindings is simply discard.
Author(s): Laurent Hubert

val merge : ('a -> 'a -> 'a) -> 'a t -> 'a t -> 'a t
merge f m1 m2 returns a map that has the bindings of m1 and m2 and which binds k to f d1 d2 if m1 and m2 binds the same k to different d1 and d2, respectively. If d1 equals d2, f d1 d2 is supposed to return d1.
Author(s): Laurent Hubert
